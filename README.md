# ITC-Graficacion-Workspace

Welcome to the **ITC-Graficacion-Workspace**! This repository contains projects, examples, and exercises for the subject of Computer Graphics ("Graficaci√≥n") using [p5.js](https://p5js.org/).

These projects showcase various graphical techniques and game implementations explored during the course.

---

## Getting Started

To run the sketches:

1. Make sure you have [p5.js](https://p5js.org/) included in your HTML file.
2. Create a blank sketch and copy the project code into it.
3. Open the sketch in your browser to see the demo!
